 ABA问题
 假设current_value=1 , A线程在进行cas判断前，B线程拿到后先改成2 ，在改成1 ，那么此时A线程就通过cas判断。
 产生的后果：
    如果是基本数据类型，没问题；
    如果是引用数据类型：可能存在问题。
 解决方式，增加一个版本号，每次修改，版本号都+1 ，判断的时候不仅判断值，也同时判断版本号（AtomicStampedReference）
